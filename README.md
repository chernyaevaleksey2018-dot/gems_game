Чтобы запустить файл нужно создать директорию build в папке проекта
```bash
mkdir build
cd build
```
Так же необходимо иметь установленную в системе систему сборки cmake для того, чтобы можно было собрать проект
```bash
cmake ..
```
Далее в зависимости от системы сгенерируются нужные файлы для запуска, для unix операционных систем будет сгенерирован `Makefile`, а для Windows файл проекта visual studio. 
Для unix-подобной системы:
```bash
make
./SFMLGAME
```
В VIsual Studio же все можно сделать открыв там проект и запустить все через графический интерфейс
